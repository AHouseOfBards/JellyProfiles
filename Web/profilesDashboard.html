⚠️ Client Script Auto-Injection Failed The plugin was unable to automatically modify Jellyfin's index.html due to folder/file write permissions on your host system. Without this script, the "Who's Watching?" profile gate and switcher will not load.

How to fix: Please run the appropriate command for your system:
  • Linux (Native): Run sudo chown -R jellyfin:jellyfin /usr/share/jellyfin/web then restart Jellyfin.
  • Docker: Run docker exec -u root <container-name> sed -i 's|</body>|<script src="/plugins/profiles/profiles.js" defer></script>\n</body>|' /jellyfin/jellyfin-web/index.html from the host.
  • Windows (Admin Command Prompt): Run icacls "C:\Program Files\Jellyfin\Server\jellyfin-web\index.html" /grant "NT AUTHORITY\NetworkService:(M)" then restart Jellyfin.

Bonfire/JellyProfiles Settings

ⓘ  Where to create & manage profiles Profiles are created, edited, and deleted from the Jellyfin home screen. After logging in, look for the Switch Profile button in the top navigation bar (a person icon next to the search and account buttons). Clicking it opens the profile manager where you can add sub-profiles, set PINs, pick avatar colors, and configure library access & parental controls.

This settings page is for server administrators only and controls the maximum number of profiles allowed per account and whether a master PIN is required before creating new sub-profiles.
